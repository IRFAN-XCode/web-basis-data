Apa itu Primary Key?

Primary Key adalah atribut atau kumpulan atribut yang digunakan untuk mengidentifikasi setiap baris data dalam sebuah tabel secara unik. Primary Key dipilh karena keunikan, sehingga dapat dibedakan dengan baris data yang lain. Primary Key berfungsi sebagai tanda pengenal dan penghubung antar tabel dalam suatu relational database. Primary Key juga sangat penting dalam proses pencarian data dan memastikan integritas antar data pada tabel.
Primary Key dapat dipilih dari satu atau kumpulan atribut yang memenuhi syarat-syarat tertentu, seperti harus berisi nilai unik, tidak boleh berisi nilai kosong, dan harus berdiri sendiri pada tabel atau atomik. Dalam ERD, Primary Key direpresentasikan dengan simbol kunci atau underline pada atribut yang dipilih sebagai Primary Key. Primary Key juga dapat digunakan sebagai Foreign Key pada tabel lain untuk menghubungkan antar tabel.(Source)

Apa itu Foreign Key?

Foreign Key atribut atau kumpulan atribut yang digunakan untuk menghubungkan antara tabel master dengan tabel turunannya. Foreign Key digunakan untuk menciptakan hubungan antara dua tabel dalam database relasional. Foreign Key juga dikenal sebagai kunci tamu karena merupakan kunci yang terdapat pada tabel kedua atau child. Foreign Key merujuk pada suatu kolom yang unik untuk menghubungkan antar tabel.
Dalam ERD, Foreign Key direpresentasikan dengan simbol kunci yang terhubung dengan Primary Key pada tabel lain. Foreign Key juga dapat digunakan sebagai Primary Key pada tabel turunan untuk menghubungkan antar tabel.(Source)

Apa itu Candidate Key?

Candidate Key adalah kumpulan atribut yang dapat membedakan setiap baris data dalam sebuah relasi secara unik. Candidate Key juga dikenal sebagai kunci calon karena merupakan kunci yang memenuhi syarat untuk menjadi Primary Key. Candidate Key berfungsi sebagai tanda pengenal dan penghubung antar tabel dalam suatu relational database.
Dalam ERD, Candidate Key direpresentasikan dengan simbol underline pada atribut yang memenuhi syarat untuk menjadi Primary Key. Candidate Key juga dapat digunakan sebagai Foreign Key pada tabel lain untuk menghubungkan antar tabel.(Source)
